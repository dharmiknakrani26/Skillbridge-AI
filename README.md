# SkillBridge AI

SkillBridge AI is a human-led, Gemini-powered resume and job fit review app for entry-level job seekers.

The app helps users upload or paste a resume, compare it with a job description, review skill gaps, and download a PDF improvement report.

## Built For

Build with Gemini XPRIZE

## Features

- Resume upload support for PDF, DOCX, TXT, and Markdown
- Resume text input
- Job description input
- Gemini-powered fit analysis
- Job fit score
- Matching skills review
- Missing skills review
- Resume improvement suggestions
- Project recommendation
- 7-day action plan
- PDF review report download

## Tech Stack

- Python
- Streamlit
- Gemini API
- Google GenAI SDK
- Pandas
- pypdf
- python-docx
- ReportLab

## Human-Led + Gemini-Powered

SkillBridge AI is human-led and Gemini-powered.

Human-led work includes:
- Product idea
- User flow
- Design choices
- Testing
- Screenshots
- Final submission planning

Gemini-powered work includes:
- Resume and job description analysis
- Fit score explanation
- Skill gap review
- Resume improvement suggestions
- 7-day action plan

## Safety Note

The Gemini API key is stored locally in a `.env` file.

The `.env` file is protected by `.gitignore` and should not be uploaded to GitHub.
